# Pi-uart-to-mqtt

Pi: Transfer date from UART to MQTT

Date: Sep 6, 2016
Author: sangseu


+ Setup Raspbian OS

OS: Jessie lite
SSH:

$ ssh pi@IP

$ sudo apt-get update

+ Install pip

Download:

$ wget https://bootstrap.pypa.io/get-pip.py

Install:

$ sudo python get-pip.py

+ Install pySerial

$ sudo python -m pip install pyserial

+ Install paho-mqtt

$ sudo pip install paho-mqtt
